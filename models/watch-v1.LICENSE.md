# Watch model attribution

`watch-v1.glb` is the “Tag Heuer Monaco X GULF Watch” model by ar-watches.

- Source: https://sketchfab.com/3d-models/tag-heuer-monaco-x-gulf-watch-31ba7140fb3146b2af3cbdc13022421c
- Demo adaptation: https://github.com/pmndrs/examples/tree/main/demos/bouncy-watch
- License: Creative Commons Attribution 4.0 International (CC BY 4.0)
  https://creativecommons.org/licenses/by/4.0/

The model is stored locally so it can be downloaded, edited in a GLB-compatible
tool such as Blender, and reloaded through the 3D page’s “Replace GLB” control.
